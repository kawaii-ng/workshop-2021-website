# 2021 Webpage for Teaching Scratch

## Features

- Provide the link to the important website
- Provide the demo for children to play with
- Provide the materials for downloading
- Upload the screen capture of the exercise in a convenient way.

## Usage

To running the react app, it can be enter the command below to the command line. 

cd workshop-workpage
npm i
npm start
